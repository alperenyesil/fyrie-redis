net.fyrie.redis

Command

class Command [A] extends Product

Creates commands to send to a Redis server, performing any needed serialization with the given Format and provides a Handler for the server's response. Implementing classes should either be case classes or case objects to allow simple formatting of parameters with very little extra code.
